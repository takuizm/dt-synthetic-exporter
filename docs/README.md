# 活用方法

## 技術者向け

### **技術深掘り**
1. `analysis/20250527_メトリクス技術分析_v1.md` でAPI制約を理解
2. `analysis/20250527_メトリクス詳細比較_v1.py` で分析手法を学習
3. `development/20250527_プロジェクト開発履歴_v1.md` で実装経緯を把握

### **運用設計**
1. `analysis/20250527_実行結果レポート_v1.md` で実績を確認
2. 本番環境への展開計画を策定
3. 監視項目・閾値の設定参考

### **問題解決**
1. 技術的課題は `analysis/` フォルダを参照
2. 開発経緯は `development/` フォルダを確認
3. 類似問題の解決事例を活用

## 新規参加者向け

### **学習フロー**
1. メインの `README.md` で全体把握
2. `docs/クイックスタートガイド_20250126_v1.md` で実行体験
3. `analysis/20250527_実行結果レポート_v1.md` で実績確認
4. 必要に応じて技術詳細を `analysis/` で学習
